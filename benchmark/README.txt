The benchmarks in this folder consist of collections of XPath queries
extracted automatically from XSLT and XQuery files using XPParser
(http://git.lsv.fr/schmitz/xpparser).

The sources used for constructing the benchmark were from the
following open-source projects:

  - DocBook XSL (http://docbook.sourceforge.net/)
  - TEI XSL (https://github.com/TEIC/Stylesheets)
  - HTML Book (https://github.com/oreillymedia/HTMLBook)
  - XQuery parser (https://github.com/jpcs/xqueryparser.xq)
  - eXist-db (https://github.com/eXist-db)
  - HisTEI (https://github.com/odaata/HisTEI)
  - transform.xq (https://github.com/jpcs/transform.xq)
  - ml-enrich (https://github.com/freshie/ml-enrich)
  - xquerydoc (https://github.com/xquery/xquerydoc)
  - openinfoman (https://github.com/openhie/openinfoman)
  - Oxford Dict API (https://github.com/AdamSteffanick/od-api-xquery)
  - MarkLogic Commons (https://github.com/marklogic/commons)
  - datascience (https://github.com/adamfowleruk/datascience)
  - Link Management BaseX (https://github.com/dita-for-small-teams/dfst-linkmgmt-basex/)
  - Semantic Web (https://github.com/HeardLibrary/semantic-web/)
  - eXist annotation store (https://github.com/telic/exist-annotation-store/)
  - xqtest (https://github.com/irinc/xqtest/)
  - data.xq (https://github.com/jpcs/data.xq/)
  - graphxq (https://github.com/apb2006/graphxq/)
  - ml-invoker (https://github.com/fgeorges/ml-invoker/)
  - treedown (https://github.com/biblicalhumanities/treedown/)
  - XQJSON (https://github.com/joewiz/xqjson/)
  - fots BaseX (https://github.com/LeoWoerteler/fots-basex/)
  - GPXQuery (https://github.com/dret/GPXQuery/)
  - rbtree.qx (https://github.com/jpcs/rbtree.xq/)
  - xquery-libs (https://github.com/adamretter/xquery-libs/)
  - Guid-O-Matic (https://github.com/baskaufs/guid-o-matic/)
  - functional.xq (https://github.com/jpcs/functional.xq/)
