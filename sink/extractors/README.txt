The stylesheets in this directory extract examples from the source files
of various qtspecs documents. (It's useful to check that these examples
conform to the syntax also defined by these documents.)

The stylesheet...      extracts...                 from...
---------------------  --------------------------  -----------------------
xpath_sp.xsl           XPath exprs                 the XPath/XQuery spec
xquery_sp.xsl          XQuery queries              the XPath/XQuery spec
xquery_uc.xsl          XQuery queries              the XQuery UC
fulltext_ucsp.xsl      XQuery Full Text queries    the Full Text UC + spec
fulltext_sp_other.xsl  schemas and XQuery queries  the Full Text spec
scripting_uc.xsl       XQuery Scripting queries    the Scripting UC

"UC" = "Use Cases doc"
